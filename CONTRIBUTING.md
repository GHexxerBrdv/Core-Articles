# Contributing to Core-Articles

First off — thank you for considering a contribution! ❤️  
Your time and effort are deeply appreciated.

---

## 🚀 Quick Reference

I want to...

- 🐛 _Report a outdated or wrong information_ ➡️ [Open an issue](#issues)
- ✨ _Add a new article_ ➡️ [Open an issue first to discuss, then submit a pull request](#issues)
- 🔄 _Update existing articles_ ➡️ [Open an issue first to discuss, then submit a pull request](#issues)
- 💻 _Refactor existing articles_ ➡️ [Create a pull request](#pull-requests)

---

## 📂 Issues

Found a wrong infromation or have a suggestion?  
Please open an issue [here](https://github.com/GHexxerBrdv/Core-Articles.git).

In your issue, be sure to include:
- A clear description of topic of article and valid references
- Updated content with proper references in case of wrong information or outdated content in articles

---

## 🔁 Pull Requests

You're welcome to open a pull request directly!  
However, for anything beyond small fixes, **please open an issue first to discuss your idea** 🙏🏼

### How to contribute:
1. Fork the repository
2. Create a new branch (`git checkout -b my-feature`)
3. Commit your changes
4. Submit a pull request

If you need help with Git or the process, feel free to ask 🤗  
Note: Reviews may take time — your patience is appreciated! 😇
