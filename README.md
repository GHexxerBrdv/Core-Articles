# Order of readings

> **Note:** Each topic (eg. Consensus layer, EVM etc) is independent and can be read in any order. But in each topic the order of reading article is important which is mentioned below.

## Consensus layer

- BeaconChain - An overview of the whole beacon chain, it's components etc.
- LMD-GHOST - A deep dive into the LMD GHOST rule, to find the head of the chain correcly. it includes the algorithm and how it works.


## EVM 

- EVM - A deep dive into the EVM and it's working under then hood in the execution layer.
- Stack EVM - Explanation of the stack data structure used in the EVM.
- Memory EVM - Explanation of the memory model used in the EVM.
- Storage EVM - Explanation of the storage model used in the EVM.
- EVM working example - Explanation of the working of EVM by executing simple bytecode. which includes the usage of stack, memory and storage and calldata.
- GAS EVM - Explanation of the gas model and how it works in the EVM.


## Transactions

- Transaction POS - Explanation of the execution of a transaction in the ethereum POS.
